HTML Anchor Links

How to link elements in the same page in HTML

Example of an anchor link

Link to anchor on same page

Code example:

<a href="#example">your link</a>

When clicking or tapping on the link above, the browser will jump to the heading below with this code:

<h5><a id="#example">your link</a></h5>
